# MediMate: Proactive Wellness Agent via LLMs

**Built for GenAI Genesis 2026 Hackthon** 🚀
MediMate isn't just another reminder app,it's a context-aware health steward. By leveraging LLMs, MediMate connects the dots between your medication adherence, physical activity, and nutritional needs to provide personalized, real-time wellness guidance. It can also be used as tracker for caregivers and family members.
